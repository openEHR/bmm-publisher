2+^h|*RESOURCE_DESCRIPTION*

2+a|Defines the descriptive meta-data of a resource.

2+h|*Attributes*

|*original_author*: `xref:BASE:foundation_types:structure_types.adoc#_hash_class[Hash]<xref:BASE:foundation_types:primitive_types.adoc#_string_class[String],xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]>` [1..1]
a|Original author of this resource, with all relevant details, including organisation.

|*other_contributors*: `xref:BASE:foundation_types:structure_types.adoc#_list_class[List]<xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]>` [0..1]
a|Other contributors to the resource, probably listed in  name <email>  form.

|*lifecycle_state*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [1..1]
a|Lifecycle state of the resource, typically including states such as: initial, submitted, experimental, awaiting_approval, approved, superseded, obsolete.

|*resource_package_uri*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [0..1]
a|URI of package to which this resource belongs.

|*other_details*: `xref:BASE:foundation_types:structure_types.adoc#_hash_class[Hash]<xref:BASE:foundation_types:primitive_types.adoc#_string_class[String],xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]>` [0..1]
a|Additional non language-sensitive resource meta-data, as a list of name/value pairs.

|*parent_resource*: `xref:RM:common:resource_package.adoc#_authored_resource_class[AUTHORED_RESOURCE]` [1..1] +
`{default{nbsp}={nbsp}0..1}`
a|Reference to owning resource.

|*details*: `xref:BASE:foundation_types:structure_types.adoc#_hash_class[Hash]<xref:BASE:foundation_types:primitive_types.adoc#_string_class[String],xref:RM:common:resource_package.adoc#_resource_description_item_class[RESOURCE_DESCRIPTION_ITEM]>` [1..1]
a|Details of all parts of resource description that are natural language-dependent, keyed by language code.

2+h|*Invariants*

2+a|__Original_author_valid__: `not original_author.is_empty`

2+a|__Lifecycle_state_valid__: `not lifecycle_state.is_empty`

2+a|__Details_valid__: `not details.is_empty`

2+a|__Language_valid__: `parent_resource /= Void implies details.for_all (d &#124; parent_resource.languages_available.has (d.language.code_string))`

2+a|__Parent_resource_valid__: `parent_resource /= Void implies parent_resource.description = self`
