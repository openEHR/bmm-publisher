2+^h|*P_ARCHETYPE_HRID*

2+a|Human_readable identifier (HRID) for an archetype or template.

2+h|*Attributes*

|*namespace*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [0..1]
a|Reverse domain name namespace identifier.

|*rm_publisher*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [1..1]
a|Name of the Reference Model publisher.

|*rm_package*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [1..1]
a|Name of the package in whose reachability graph the rm_class class is found (there can be more than one possibility in many reference models).

|*rm_class*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [1..1]
a|Name of the root class of this archetype.

|*concept_id*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [1..1]
a|The short concept name of the archetype as used in the multi-axial archetype_hrid.

|*release_version*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [1..1]
a|The full numeric version of this archetype consisting of 3 parts, e.g. 1.8.2. The archetype_hrid feature includes only the major version.

|*version_status*: `xref:BASE:base_types:definitions_package.adoc#_version_status_enumeration[VERSION_STATUS]` [1..1]
a|The status of the version, i.e. released, release_candidate etc.

|*build_count*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [1..1]
a|The build count since last increment of any version part.
