2+^h|*__EXPR_OPERATOR (abstract)__*

2+a|Abstract parent of operator types.

2+|Inherits: xref:AM:AOM1.4:archetype_package.adoc#_expr_item_class[EXPR_ITEM]

2+h|*Attributes*

|xref:AM:AOM1.4:archetype_package.adoc#_expr_item_class[EXPR_ITEM].*type*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [1..1]
a|Type name of this item in the mathematical sense. For leaf nodes, must be the name of a primitive type, or else a reference model type. The type for any relational or boolean operator will be “Boolean”, while the type for any arithmetic operator, will be “Real” or “Integer”.

|*precedence_overridden*: `xref:BASE:foundation_types:primitive_types.adoc#_boolean_class[Boolean]` [0..1]
a|True if the natural precedence of operators is overridden in the expression represented by this node of the expression tree. If True, parentheses should be introduced around the totality of the syntax expression corresponding to this operator node and its operands.

|*operator*: `xref:AM:AOM1.4:archetype_package.adoc#_operator_kind_enumeration[OPERATOR_KIND]` [1..1]
a|Code of operator.
