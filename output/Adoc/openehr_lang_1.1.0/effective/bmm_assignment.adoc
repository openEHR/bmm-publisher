2+^h|*BMM_ASSIGNMENT*

2+a|Statement type representing an assignment from a value-generating source to a writable entity, i.e. a variable reference or property.

At the meta-model level, may be understood as an initialisation of an existing meta-model instance.

2+|Inherits: xref:LANG:bmm:statement_package.adoc#_bmm_statement_item_class[BMM_STATEMENT_ITEM], xref:LANG:bmm:statement_package.adoc#_bmm_statement_class[BMM_STATEMENT], xref:LANG:bmm:statement_package.adoc#_bmm_simple_statement_class[BMM_SIMPLE_STATEMENT]

2+h|*Attributes*

|*target*: `xref:LANG:bmm:expression_package.adoc#_el_value_generator_class[EL_VALUE_GENERATOR]` [1..1]
a|The target variable on the notional left-hand side of this assignment.

|*source*: `xref:LANG:bmm:expression_package.adoc#_el_expression_class[EL_EXPRESSION]` [1..1]
a|Source right hand side) of the assignment.

2+h|*Invariants*

2+a|__Inv_target_writable__: `target.is_writable`
