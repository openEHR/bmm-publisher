2+^h|*BMM_PARAMETER_TYPE*

2+a|Definition of a generic parameter in a class definition of a generic type.

2+|Inherits: xref:LANG:bmm:core_package.adoc#_bmm_type_class[BMM_TYPE], xref:LANG:bmm:core_package.adoc#_bmm_unitary_type_class[BMM_UNITARY_TYPE]

|*name*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [1..1]
a|Name of the parameter, e.g. 'T' etc. The name is limited to 1 character and upper-case.

|*type_constraint*: `xref:LANG:bmm:core_package.adoc#_bmm_effective_type_class[BMM_EFFECTIVE_TYPE]` [0..1]
a|Optional conformance constraint that must be the name of a defined type.

|*inheritance_precursor*: `xref:LANG:bmm:core_package.adoc#_bmm_parameter_type_class[BMM_PARAMETER_TYPE]` [0..1]
a|If set, is the corresponding generic parameter definition in an ancestor class.
