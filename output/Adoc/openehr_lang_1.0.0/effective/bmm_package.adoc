2+^h|*BMM_PACKAGE*

2+a|Abstraction of a package as a tree structure whose nodes can contain other packages and classes.

The `_name_` may be qualified if it is a top-level package.

2+|Inherits: xref:LANG:bmm:core_package.adoc#_bmm_declaration_class[BMM_DECLARATION], xref:LANG:bmm:core_package.adoc#_bmm_package_container_class[BMM_PACKAGE_CONTAINER]

2+h|*Attributes*

|xref:LANG:bmm:core_package.adoc#_bmm_declaration_class[BMM_DECLARATION].*name*: `link:/classes/String[String] ` [1..1]
a|Name of this model element.

|xref:LANG:bmm:core_package.adoc#_bmm_declaration_class[BMM_DECLARATION].*documentation*: `link:/classes/Hash[Hash] <link:/classes/String[String] ,link:/classes/Any[Any] >` [0..1]
a|Optional documentation of this element, as a keyed list.

It is strongly recommended to use the following key /type combinations for the relevant purposes:

* `"purpose": String`
* `"keywords": List<String>`
* `"use": String`
* `"misuse": String`
* `"references": String`

Other keys and value types may be freely added.
